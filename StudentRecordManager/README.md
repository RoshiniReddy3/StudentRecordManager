# Student Record Manager (Java)

A simple console-based Java application to manage student records.

## Features
- Add new student with ID, name, course and grade
- View all students
- Search student by ID
- Demonstrates Object-Oriented Programming and use of Java Collections

## Technologies Used
- Core Java
- ArrayList
- Basic console input/output

## How to Run
1. Open a terminal in the `src` folder.
2. Compile the code:

   ```bash
   javac Main.java Student.java StudentService.java
